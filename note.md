### Hello Cargo

Cargo 是 Rust 的构建系统和包管理器
- cargo new hello_cargo(创建项目)    执行后没有出现git：父目录有了git文件
- cargo init(将项目转化为cargo项目)
- cargo build(构建项目)
- ./target/debug(存放可执行文件)
- Cargo.lock(记录实际依赖，不用维护)
- cargo run(构建+运行)
- cargo check(确保编译)
